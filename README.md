## Setup
To run this project, install it locally using npm:

##### Server
```
$ cd backend
$ npm install
$ npm start
```

##### Web
```
$ cd frontend
$ npm install
$ npm start
```

##### Mobile
```
$ cd mobile
$ npm install
```

> Acept expo instalation :warning:

```
$ npm start
```
